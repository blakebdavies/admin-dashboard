 <h2>BASH | Admin Dashboard</h2>
 <p> This is a project focused on creating a simple, visually engaging and responsive, Admin Dashboard for an Ecommerce Camera Website using HTML, CSS, and JavaScript with light and dark mode theme functionality.</p>
 <h2> Technologies and Concepts</h2>
 <ul>
  <li>Vanilla Javascript</li>
  <li>Light/Dark Mode Toggle Theme</li>
  <li>Responsive Design</li>
  <li>CSS Variables</li>
  <li>CSS Grid</li>
  <li>CSS Flexbox</li>
  <li>CSS transitions and animations</li>
  <li>Media Queries</li>
  </ul>
